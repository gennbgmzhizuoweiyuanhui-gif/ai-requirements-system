AI要件定義システム - Windows版 使い方
========================================

【起動方法】
1. ai-requirements-system.exe をダブルクリックして起動します
2. 黒いコンソール画面が開き、サーバーが起動します
3. ブラウザで http://localhost:3001 を開きます
4. AIプロバイダー（Claude または Gemini）を選択し、APIキーを入力して使用します

【終了方法】
- コンソール画面を閉じるか、Ctrl+C を押してサーバーを停止します

【必要なもの】
- Anthropic APIキー（Claudeを使用する場合）
  取得先: https://console.anthropic.com/
- Google APIキー（Geminiを使用する場合）
  取得先: https://aistudio.google.com/

【注意事項】
- EXEファイルと同じフォルダに index.html および icon.png を配置してください
- インターネット接続が必要です（API通信のため）
- APIキーはブラウザのlocalStorageに保存されます（このPC内のみ）

【トラブルシューティング】
- 「ポートが使用中」エラーが出る場合: 既に起動中のEXEがないか確認してください
- ブラウザで開けない場合: コンソール画面にエラーが表示されていないか確認してください
