# AI要件定義システム 設計書

**日付:** 2026-04-18  
**バージョン:** 1.0  
**用途:** 2026年4月22日 中田組様（北海道稚内）デモ向けプロトタイプ

---

## 概要

現場の曖昧な発話をAIが精密な要件定義書に変換するWebアプリ。  
単一HTMLファイルで動作し、インストール不要・ブラウザで即起動。

---

## アーキテクチャ

- **ファイル構成:** `index.html` 1ファイル完結
- **外部依存:** ゼロ（CDN不使用、オフライン対応可）
- **CSS:** `<style>` タグにインライン記述
- **JS:** `<script>` タグにインライン記述
- **MarkdownレンダリングL** 自前実装（正規表現ベース）

---

```
┌─────────────────────────────────────┐

│  ┌───────────────────────────────┐  │
```

---

## 機能要件

1. **音声入力:** Web Speech API使用。ボタンを押している間だけ録音。離したら自動でテキスト変換してテキストエリアに追記。
2. **テキスト入力:** 直接キーボード入力も可能。音声入力との併用可。
3. **要件定義書生成:** Claude API（`claude-sonnet-4-20250514`）に送信し、6セクションのMarkdownを取得。
4. **整形表示:** 自前MarkdownパーサーでHTML変換して表示（見出し・箇条書き・太字）。
5. **Markdownコピー:** コピーボタンでMarkdown原文をクリップボードにコピー。
6. **APIキー入力欄:** ヘッダーにinputフィールド。ハードコードしない。

---

## 出力フォーマット（Claudeへの指示）

Claude APIへのシステムプロンプトで以下6セクションのMarkdown出力を強制：

```
## システム名
## 目的・背景
## 機能要件
## 非機能要件
## 用語定義
## 備考・確認事項
```

---

## 自前Markdownパーサー仕様

| Markdown記法 | HTML変換先 |
|---|---|
| `## 見出し2` | `<h2>` |
| `### 見出し3` | `<h3>` |
| `- 箇条書き` | `<ul><li>` |
| `**太字**` | `<strong>` |
| `---` | `<hr>` |
| 空行 | `<p>` 区切り |

---

## エラー処理

- APIキー未入力 → 日本語アラートメッセージ
- APIエラー（4xx/5xx） → エラー内容を出力エリアに日本語で表示
- ネット切断 → エラーメッセージを出力エリアに表示
- 音声API非対応ブラウザ（Firefox等） → 音声ボタンを非表示にしてテキスト入力のみに切り替え
